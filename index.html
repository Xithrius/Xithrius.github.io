Commands

Weather
Command Info Arguments Example
daily Returns a graph of the next days defined by the user. ZIP (zip/postal code) | DAYS (amount of days to be included in the list, default 7) | COUNTRY (default to US) .weather daily 12345
init Initiates a background task to record weather data in a specific area. ZIP (zip/postal code) | COUNTRY (default to US) .weather init 12345
Reddit
Command Info Arguments Example
top Returns link(s) from the top of all time of a subreddit. SUBREDDIT (specific subreddit) | AMOUNT (amount of links, default 1, max 5) .reddit top 2
hot Returns link(s) from the hot of all time of a subreddit. SUBREDDIT (specific subreddit) | AMOUNT (amount of links, default 1, max 5) .reddit hot 5
